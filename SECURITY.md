# Security Policy

## Supported Versions

Security updates are provided for the latest published minor release in the `0.x` line.

| Version | Supported |
| ------- | --------- |
| 0.x     | Yes       |
| < 0.x   | No        |

## Reporting a Vulnerability

Please do not report security vulnerabilities in public issues.

Use GitHub Security Advisories to report privately:

https://github.com/qonhq/qon-js/security/advisories/new

Include the following details when possible:

- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix or mitigation

We will acknowledge reports as quickly as possible and work on remediation based on severity.
