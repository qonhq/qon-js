# Support

## Getting Help

- Ask questions in GitHub Discussions:
  https://github.com/qonhq/qon-js/discussions
- Report bugs using issue templates:
  https://github.com/qonhq/qon-js/issues/new/choose

## Security Issues

For security reports, use private advisories only:

https://github.com/qonhq/qon-js/security/advisories/new

## Scope

Support covers the `qon-js` package and its integration with the Qon core bridge mode.
